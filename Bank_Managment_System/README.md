# 🏦 Bank Management System

A simple command-line Bank Management System built with Python.

This project was created as part of **Project Build365** to practice Python fundamentals, file handling, functions, and exception handling.

---

## 🚀 Features

- Create New Account
- Deposit Money
- Withdraw Money
- Check Account Balance
- Save Account Information
- Input Validation
- Exception Handling
- Menu-Driven Interface

---

## 📚 Concepts Used

- Python Functions
- Conditional Statements
- Loops
- File Handling
- Exception Handling
- User Input Validation
