# 🚀 Project Build365 | Day 13 - NumPy Fundamentals

## 📌 Overview

Welcome to **Day 13** of my **Project Build365** learning journey.

Today, I started learning **NumPy**, one of the most important libraries in Python for numerical computing and the foundation of Data Science, Machine Learning, and Artificial Intelligence.

---

## 🎯 Objective

The goal of this project is to understand the fundamentals of NumPy, learn how to create and work with arrays, and explore the basic functions that make numerical operations faster and more efficient than Python lists.

---

## 📚 Topics Covered

- Introduction to NumPy
- Why NumPy?
- NumPy vs Python Lists
- Importing NumPy
- Creating Arrays using `np.array()`
- 1D Arrays
- 2D Arrays
- Array Properties
  - `.shape`
  - `.ndim`
  - `.size`
  - `.dtype`
- Special Array Functions
  - `np.zeros()`
  - `np.ones()`
  - `np.arange()`
  - `np.linspace()`
- Type Conversion using `.astype()`
